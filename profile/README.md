# Voxa

A hands-free voice line to [Claude Code](https://claude.com/claude-code). Run
`npx voxa-code` on your laptop, pair your phone with a QR code, and talk to
Claude the way you'd talk to a person.

- **[voxa-code](https://github.com/Ti-03/voxa)** — the open-source CLI/server (MIT), published as
  [`voxa-code`](https://www.npmjs.com/package/voxa-code) on npm and
  [PyPI](https://pypi.org/project/voxa-code/).
- **voxa-ios** — the companion iPhone app (private).
- **voxa-cloud** — the hosted relay + billing service behind `api.voxa.space` (private).
- **voxa-web** — the marketing site, [voxa.space](https://voxa.space) (private).

Website: [voxa.space](https://voxa.space) · Install: `npx voxa-code`
